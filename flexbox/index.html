Flex items want to be as small as possible while being on one line. Intrinsic sizing, if items can fit within parent w/o line breaks. If runs out of room, line breaks. Flex-grow (default is 0), flex-shrink (default is 1).

Justify content: The CSS justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container.

Flex shorthand

flex: flex-grow (1) flex-shrink (1) flex-basis (auto)

The flex-basis CSS property sets the initial main size of a flex item. It sets the size of the content box unless otherwise set with box-sizing.

align-items controls alignment of items on the cross axis in Flexbox.
